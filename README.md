### 🔭 What I'm Working On
- **ICS Anomaly Detection (Graduation Research):** Built and benchmarked a real-time Digital Twin platform (Next.js, Three.js, Modbus TCP/OPC UA) evaluating five unsupervised ML models against IEC 62443-aligned attack scenarios — top result: a 71.6% F1-score detecting stealthy ±2.5σ sensor spoofing attacks.
- **Bio-Mimetic Robotic Arm:** Engineering a 5-DOF robotic arm with EMG signal classification (Random Forest, 94% accuracy) and inverse kinematics for real-time biomimetic control.
- **Barrier-Free Accessibility (TEKNOFEST 2025 Finalist):** A real-time, browser-based pipeline translating live spoken audio into 3D Sign Language animations, using WebSockets and OpenAI Whisper.
- **PardusGuard:** An open-source system hardening and security assessment tool for Pardus Linux, built with Python and GTK4.
